# YGO Practice Lab

A standalone Yu-Gi-Oh goldfishing / deck-testing tool with a real duel engine
(ygopro-core / ProjectIgnis card scripts), built with React + Vite.

Live: enabled via GitHub Pages (Actions workflow).

## Dev
```
npm install
npm run dev
```

Engine, card database, and card scripts are pinned to fixed ProjectIgnis
commits so the build stays reproducible.
